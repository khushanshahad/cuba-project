Core HTML Elements

HTML (HyperText Markup Language) is the backbone of any website. Here are the fundamental concepts:

  • Document Structure: Every page needs a <!DOCTYPE html>, <html>, <head>, and <body>.
  • Headings: Used from <h1> (most important) down to <h6>.
  • Paragraphs: Wrapped in <p> tags.

💡 Resource: Check out the MDN HTML Web Docs for deep dives.

Hyperlinks & Multimedia

Connecting pages and embedding media is what made the web famous.

  • Anchor Tags: Used for links. Example: <a href="url">Link Text</a>.
  • Attributes: Use target="_blank" to open links in a new tab.
  • Images: Embedded using <img src="image.jpg" alt="description"> (Note: it's a self-closing tag!).

💡 Resource: Learn more about HTML Links at W3Schools.

Forms & Layouts

Gathering user input and structuring complex data formats.

  • Forms: The <form> tag holds inputs, textareas, and submit buttons.
  • Tables: Built using <table>, rows <tr>, headers <th>, and data cells <td>.
  • Semantic Layouts: Modern HTML uses <header>, <nav>, <main>, and <footer> for better SEO.
