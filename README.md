# Cuba Information Repository

A curated research reference on Cuba — covering U.S.-Cuba relations, sanctions, healthcare, population health, medical innovations, and the humanitarian impact of the U.S. blockade.

Compiled May 2026. Sources are linked directly in the knowledge bank.

## Sources

All sources are cited inline with direct hyperlinks. Source types include:

- U.S. government documents (executive orders, OFAC, DOJ, Federal Register)
- UN and international bodies (OHCHR, WHO, UN News)
- Peer-reviewed journals (Lancet, BMJ Global Health, Frontiers in Immunology, Journal of Alzheimer's Disease)
- Investigative and international journalism (The New Humanitarian, Le Monde, Politico, The Conversation, Science)
- Legal and regulatory documents (EU Blocking Statute, Visa Core Rules, CACR)
- Think tanks and research centers (CEPR, Council on Foreign Relations, National Security Archive)

## How to Use

The knowledge bank is a single HTML file (`index.html`) that runs entirely in the browser — no installation, no dependencies. Open it locally or visit the GitHub Pages link above.
